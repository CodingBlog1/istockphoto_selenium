### Introduction

```
Scrape the image urls from the Unsplash web with the help of multiprocessing 
```

### Setup virtual environment 

```
for linux:

    python -m venv venv 
    source/bin/activate

for windows:

    python -m venv venv 
    venv/Scripts/activate
```

### install the dependencies 

```
pip install -r requirmeents.txt
```

### Run the main.py file 

```
python main.py
```
